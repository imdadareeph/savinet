Welcome to SAVINET

SoorejAnkitVidyaImdadNET

The SAVINET is a Social networking Website which serves as an access point for Bloggers and anyone can create his profile, follow up other users who are registered. It is based on MVC architecture and is mostly customizable. Besides following each other, users can upload audio files, raw documents to a limited size. Also, email notification (Invitation) is available.

Recommending to install git-bash. Its much easier and simpler.

$ cd savinet
$ git clone https://github.com/imdadareeph/savinet.git

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Designing

All designs were self created and have been done using web learning. Some images and third pary JS have also been used. All will be updated in due time. Currently thats all.

Having issues with deployment?

Feel free to contact the developers. This was a very old code with no pattern. So do not complain about a bad way of doing it. Infact this was done during the learning phase of web application. A better version of it will be soon available which is done in Spring Hibernate JSF primefaces JQuery Mysql and some other Libraries.
